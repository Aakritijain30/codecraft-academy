# 🚀 CodeCraft Academy

A modern, responsive landing page for a web-development bootcamp — built with React, Vite, and Tailwind CSS.

## ✨ Features

- Responsive design — works on mobile, tablet, and desktop
- Smooth-scroll navigation between sections
- Interactive signup modal (name + email capture)
- Sections: Hero, Stats, Curriculum path, Testimonials, Pricing plans, CTA, Footer
- Clean component structure, easy to re-theme or extend

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| [Vite](https://vitejs.dev/) | Fast dev server & build tool |
| [React](https://react.dev/) | UI components |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first styling |
| [lucide-react](https://lucide.dev/) | Icon set |
